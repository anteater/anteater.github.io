Get Started

Install

Anteater is best installed via pip


$ pip install anteater

Configure

Set your filters

                  
file_contents:
  clone:
    regex: git.*clone
    desc: "Clone blocked as using an non approved external source"

Set a whitelist RegExp

                  
file_contents:
  - git clone https:\/\/github\.com\\trusted_acme\\.+
