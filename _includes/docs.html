Get Started

PIP

anteater is best installed using pip where the lastest stable release can be found.


                pip install anteater

travis-ci

To config Travis CI to use anteater.

    
	language: python

	python:
	  - "2.7"

	install:
	  - pip install anteater

	before_script:
	  - git diff --name-only HEAD^ > ./patch
	script:
	  - anteater --project antest --patchset ./patch
